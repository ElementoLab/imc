API
===

The great flexibility of ``imc`` comes from the ability to compose workflows using the API.

It provides a rich but abstract :class:`~imc.analysis.Analysis` object and implements various modules building on it depending on the data type.

In addition, the :mod:`~imc.general` module contains several analysis-independent methods and the :mod:`~imc.utils` module provides low-level functions of general use.
